# 🗺️ Visited_China_City

一个交互式中国地图应用，帮助你记录走过的每一寸土地。支持省、市、县三级精度标注，数据保存在本地浏览器中。

---

## ✨ 功能特性

- **交互地图**：支持鼠标滚轮缩放和拖拽平移
- **三级精度**：可精确到省、市、区/县级别
- **快速标记**：直接点击地图省份即可标记/取消，无需操作选择框
- **逐级联动**：市、县选择器自动从高德地图接口加载最新数据
- **本地持久化**：数据存储在浏览器 localStorage，刷新后保留
- **统计面板**：实时显示已到访省份数量及覆盖率

---

## 🚀 快速开始

```
# 安装依赖
npm install

# 启动开发服务器（默认 http://localhost:3000）
npm run dev
```
